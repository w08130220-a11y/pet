# PetLife 市場研究筆記

## 全球寵物市場規模

### 整體市場
- 2024年全球寵物護理市場：$259.37B - $346B（不同來源）
- 2025年預估：$273.42B
- 2030年預估：$500B（Bloomberg Intelligence）
- 2032年預估：$427.75B - $643.53B
- 年複合成長率（CAGR）：5.1% - 7%

### 美國市場
- 2024年寵物支出：$152B
  - 寵物食品與零食：$65.8B
  - 用品、活體動物與非處方藥：$33.3B
  - 獸醫護理與產品銷售：$39.8B

### 亞洲市場
- 中國：全球最大寵物產品市場，2025年預計達 $59B
- 日本：$15B
- 韓國：$10B
- 東南亞：2023-2028 市場從 $2.8B 成長至 $5B（CAGR 12%+）

## 寵物社交網路市場

### 市場規模
- 2025年全球寵物社交網路市場：$1.4B
- 2032年預估：$4.1B
- 成長潛力巨大

### 競爭對手分析

| 公司/APP | 估值/融資 | 特點 |
|---------|----------|------|
| Petzbe | $14.3M（pre-money） | 寵物社交媒體，連接寵物主人 |
| Pet Attix | £4.4M | 英國寵物社交平台 |
| PetsApp | $4.4M 融資 | 獸醫與寵物主人連接，20萬+寵物用戶 |
| Pawmates | - | Google Play 寵物社交網路 |
| Offleash'd | - | 社交+約會雙模式 |

### 市場機會
- 目前市場仍處於探索階�段
- 電商、社交、O2O 服務各有側重
- 消費人群年輕化趨勢明顯
- 寵物在家庭中地位提升

## 關鍵洞察

1. 全球寵物市場持續高速成長
2. 寵物社交網路市場規模相對較小但成長迅速
3. 目前缺乏具有統治地位的寵物社交平台
4. 3D 虛擬寵物技術是差異化競爭優勢
5. 訂閱制商業模式在寵物服務領域有成功案例
